이미지 넣기

이미지를 넣으려면 img 요소를 사용합니다.

src 속성에는 이미지의 경로(URL)를 적습니다.

이미지 요소에는 항상 alt 속성을 사용하여 대체 텍스트를 명시해주어야 합니다. "alt"는 "Alternative text"의 약자로, 이미지를 표시할 수 없을 때 이미지를 대신하여 나타낼 텍스트를 뜻합니다.

블럭 요소 목록

다음은 HTML에서 자주 사용되는 요소들 입니다.

  1. p : 단락 (paragragh)
  2. h1 : 큰제목 (Heading 1)
  3. ul : 순서 없는 목록 (unordered list)
  4. ol : 순서 있는 목록 (ordered list)
  5. table : 표 (table)

HTML 이란?

HTML은 하이퍼텍스트 마크업 언어(Hypertext Markup Language)의 약자로 웹 페이지를 만들 때 사용되는 문서 형식입니다.

하이퍼텍스트(Hypertext)란 하이퍼링크(hyperlink)를 담고 있는 문서를 뜻합니다.하이퍼링크는 문서들이 서로 연결되어있는 고리를 말합니다.
